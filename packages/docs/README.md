# Documentation Site

The documentation site for `power-platform-playwright-toolkit`, built with
[Nextra](https://nextra.site/) (Next.js).

**Live site:** https://microsoft.github.io/power-platform-playwright-samples/

## Development

```bash
# From repo root
rush install

# Start dev server
cd packages/docs
npm run dev
```

Visit http://localhost:3000.

## Build

```bash
# Local build
npm run build

# GitHub Pages build (with base path)
npm run build:github
```

## Generating API Reference

The `pages/reference/` directory is auto-generated by TypeDoc. Regenerate after toolkit changes:

```bash
npm run generate:api
# or
cd ../power-platform-playwright-toolkit && npm run docs
```

## Structure

```
packages/docs/
├── components/          # Shared MDX components (Cards, Features, EnvTable, etc.)
├── pages/
│   ├── index.mdx        # Home page
│   ├── guide/           # User guides (getting-started, setup, auth, etc.)
│   ├── architecture/    # Architecture docs (overview, patterns, etc.)
│   ├── reference/       # Auto-generated TypeDoc API reference
│   └── contributing.mdx # Contributing and publishing guide
├── public/              # Static assets
├── theme.config.tsx     # Nextra theme configuration
└── next.config.mjs      # Next.js configuration
```

## Adding a Guide Page

1. Create `pages/guide/my-page.mdx`
2. Add an entry to `pages/guide/_meta.ts`
3. Use MDX components from `../../components/` and `nextra/components`
